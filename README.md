## ✨ Features

* Search weather by city
* Real-time temperature display
* Weather condition display
* Humidity information
* Wind speed information
* Location information
* Last updated timestamp
* Hourly weather forecast
* Modern custom UI design

## 🛠️ Tech Stack

* Java 17
* Java Swing
* Open-Meteo API
* JSON Simple
* HttpURLConnection
* Git & GitHub

## 🏗️ Architecture

AppLauncher
↓
WeatherAppGUI
↓
WeatherService
↓
Open-Meteo API

## 🚀 Challenges

* Integrating external REST APIs
* Parsing JSON responses
* Designing a custom desktop UI
* Managing weather icons dynamically
* Handling user input and data updates
